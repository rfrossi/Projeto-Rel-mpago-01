# Projeto Relâmpago 01 - Portfólio Profissional

## Descrição
Este é um portfólio pessoal desenvolvido como parte do curso Alpha EdTech, onde participei como Aspirante a Desenvolvedor Full Stack na 7ª Turma (Selene|Aurora). O projeto apresenta um site responsivo com tema escuro, destacando habilidades em desenvolvimento web, projetos realizados e informações de contato. Dezembro/2025.

## Objetivo do Projeto (Requisitos)
O que deve ser feito:  Crie seu próprio portfólio online para exibir seus projetos e habilidades.
Requisitos Mínimos:
- Estrutura clara em HTML com uso de tags semânticas. ✅
- CSS bem-organizado, responsivo e animado, a acessibilidade deve ser levada em consideração. ✅
- Deve se utilizar um CSS puro. ✅
- Deve ter um design bem-feito e bem pensado e limpo. ✅
- Deve ser criado um protótipo utilizando Figma. ✅
- O projeto deve ser guardado em um repositório público do github e servido pelo github pages. ✅
- No repositório, deve conter um README.md documentando todo o projeto ✅
Prazo: Os projetos serão apresentados nos dias 18/12/2025 e 19/12/2025.

## Tecnologias Utilizadas
- **HTML5**: Estrutura semântica da página
- **CSS3**: Estilização responsiva, incluindo variáveis CSS para tema escuro, flexbox e grid
- **Font Awesome**: Ícones para redes sociais e elementos visuais
- **Google Fonts**: Tipografia personalizada (Outfit)
- **Scripts**: JavaScript simples para menu mobile

## Estrutura do Projeto
- `index.html`: Página principal com seções Hero, Sobre, Projetos e Footer
- `style.css`: Arquivo de estilos com design moderno e responsivo
- `img/`: Pasta contendo imagens, como logos e avatares
- `readme.md`: Documentação do projeto
